# PokeAPI Web Application

PokeAPI（https://pokeapi.co/）を使用した、初代ポケモン（図鑑番号001～493）のデータ閲覧Webアプリケーションです。

## 機能

1.  **ポケモンリスト（001-493）**
    - 初代赤・緑(第一世代)～ダイヤモンド・パール(第四世代)に登場する493匹のポケモンを表示します。
    - 左側のパネルはスクロール可能で、スムーズにポケモンを探すことができます。

2.  **詳細表示（マルチビュー）**
    - 画面を左右に分割し、左側で選択したポケモンの詳細を右側に表示します。
    - 公式アートワークの画像と共に、ポケモンの名前が表示されます。

3.  **ステータス可視化（バーチャート）**
    - ポケモンのパラメータ値（HP, こうげき, ぼうぎょ, とくこう, とくぼう, すばやさ）をバーチャートで視覚的に表示します。
    - 選択時にアニメーション付きでグラフが描画されます。

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: Flexboxを用いたレイアウト、モダンでクリーンなデザイン、変数は`:root`で管理
- **JavaScript (Vanilla)**: フレームワーク未使用。`fetch` APIを用いた非同期データ取得

## 実行方法

このディレクトリにある `index.html` をブラウザ（Chrome, Firefox, Edgeなど）で開いてください。
ローカルサーバーは必須ではありませんが、アイコンやフォントの読み込みをスムーズにするために推奨されます。

```bash
# ローカルサーバーを使用する場合の例（Python環境がある場合）
python -m http.server
# その後、 http://localhost:8000 にアクセス
```

## ファイル構成

- `index.html`: アプリケーションのメイン構造
- `style.css`: スタイルシート（レスポンシブ対応、UIデザイン）
- `main.js`: データ取得、DOM操作、イベントハンドリングロジック
